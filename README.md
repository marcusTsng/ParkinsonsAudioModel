# ParkinsonsAudioModel
A mini app that can take data from audio recordings and detect early signs of Parkinson's. 

To open the app, run main.py in the terminal. Then, open index.html in Pages on any browser, ie Google Chrome. 
> Note - dependencies in requirements.txt may have to be installed (feel free to also make use of the venv):
``` pip3 install  -r requirements.txt```
